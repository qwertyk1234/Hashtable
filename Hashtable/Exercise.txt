Требуется написать программу на языке С++.

Данные считываются из входного файла input.txt
Результат выводится в выходной файл output.txt

Необходимо

1. создать хеш-таблицу для описания значений тегов HTML .
2. реализовать функции добавления, удаления и поиска элементов в хеш-таблице.
3. в программе:
	а)считать входной файл input.txt, содержащий определения тегов HTML (не менее 25)
	б)вывести полученную хеш-таблицу в файл output.txt
	в)сделать поиск введенного  с клавиатуры тега в хеш-таблице и вывести описание значения на экран.
